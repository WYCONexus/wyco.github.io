# WYCO Hub Starter Site

This is a static website starter for the WYCO hub.

## Files
- `index.html` — homepage
- `waves.html` — media / lyrics / music
- `nexus.html` — code / bots / digital projects
- `repairs.html` — professional gateway page
- `projects.html` — active build board
- `style.css` — shared styling
- `script.js` — simple mobile menu toggle

## How to preview locally
Double-click `index.html` to open it in your browser.

## How to host for free on GitHub Pages
1. Create a GitHub account.
2. Create a new public repository.
3. Upload all files from this folder.
4. In the repository, go to **Settings** → **Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and save.
7. Your site will publish to a free GitHub Pages URL.

## Customize
- Replace placeholder text in each HTML file.
- Update the Repairs button link in `repairs.html`.
- Add images, logos, or audio embeds where needed.
- Edit colors in `style.css`.
